Серивис - сайт, показывающий погоду по заданному городу.

- пользователь вводит город
- формируется запрос к API https://openweathermap.org
- если 
-- запрос проходит неуспешно: формируется ответ пользователю о том, что что-то пошло не так
-- запрос проходит успешно: формируется строка из ответа API по погоде, дописывается совет что взять с собой
- ответ выводится пользователю
